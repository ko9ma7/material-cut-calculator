# CutFlow 자재 절단 계산기

자재 모양, 재질, 규격, 원자재 길이별로 필요한 발주량과 절단 배치를 계산하는 정적 웹서비스입니다. 같은 절단 길이라도 홀 지름·개수·위치가 다르면 서로 다른 부품으로 구분합니다.

## 주요 기능

- 구조각관, L앵글, 디귿잔넬, C형찬넬, H빔, I빔 규격 444개
- 홀 지름·개수·위치를 포함한 부품 명세서(BOM)
- 톱날 두께와 끝단 여유를 반영한 Best Fit Decreasing 절단 배치
- 규격별 발주 본수, 예상 중량, 톱날 손실, 잔재율 집계
- 브라우저 자동 저장과 마지막 결과 자동 복원
- 구버전 계산기 JSON 불러오기, 새 JSON 저장
- A4 가로 결과 인쇄, 모바일·다크 모드 지원
- 서버 전송 없이 브라우저에서만 계산

> 절단 배치는 빠르고 재현 가능한 휴리스틱 결과이며 수학적 최적해를 항상 보장하지 않습니다. 실제 발주 전 자재 공차, 절단기 조건과 단위중량을 확인하세요.

## 파일 구조

```text
index.html                   화면 구조
css/styles.css               화면 디자인과 반응형 규칙
css/print.css                A4 가로 인쇄 전용 규칙
js/app.js                    이벤트와 작업 흐름
js/ui.js                     입력 행과 결과 화면 렌더링
js/calculator.js             검증과 절단 배치 계산 엔진
js/storage.js                자동 저장, JSON 호환·백업
js/materials/                자재 모양별 규격 데이터
tests/                       계산 경계와 정적 구조 검사
.github/workflows/           GitHub Pages 자동 배포
```

## 자재 규격 수정

`js/materials/`에서 해당 모양 파일을 엽니다. 재질 아래 배열에 다음 형식으로 규격을 추가하거나 수정합니다.

```js
{ "spec": "50x50x2.3", "kgPerM": 3.34 }
```

- `spec`: 화면에 표시할 규격
- `kgPerM`: 1m당 단위중량(kg/m)

모양 파일을 추가할 때는 `js/materials/index.js`에서 import한 뒤 `materialCatalog` 배열에 넣습니다.

## 로컬 실행과 검사

ES 모듈을 사용하므로 HTML 파일을 직접 더블클릭하지 말고 로컬 서버로 엽니다.

```bash
npm test
npm run serve
```

그다음 `http://localhost:4173`을 엽니다. 외부 패키지 설치는 필요 없습니다.

## 데이터와 개인정보

입력값은 현재 브라우저의 `localStorage`에만 자동 저장되며 서버나 GitHub로 전송되지 않습니다. 다른 PC로 옮길 때는 하단의 **파일 저장**으로 JSON을 내려받고 **불러오기**로 복원합니다.
